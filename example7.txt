¿Cuales de las siguientes sentencias son equivalentes a O(N)? ¿por qué?

1. O(N + P) donde P < n/2    // -> O(n)
2. O(2N)     // -> O(n)
3. O(N + log N) // -> O(n)
4. O(N + M)  // -> O(n)

al quitar las constantes y soltar los factores no dominantes todas las sentencias tienen una complejidad de -> O(n)
